# OpenCode Plan Mode Workflow

## Activation

1. Load this file via `opencode.json` instructions
2. Confirm Plan Mode is active
3. Read canonical rules: `/.agents/rules/meta-agent.md`

## Execution Flow

### Phase 1: Discovery

- Read `/.agents/memory-bank/project-brief.md`
- Read `/.agents/memory-bank/process-overview.md`
- Read active backlog item in `/.backlog/`

### Phase 2: Interview (if needed)

- Engage human in ultra-dense dialogue
- Extract constraints, vision, requirements
- No fluff, only actionable data

### Phase 3: Planning

- Decompose task into actionable steps
- Identify blockers and dependencies
- Prepare handoff for Build Mode

### Phase 4: Handoff

Create handoff with:

- **Goal:** Clear objective
- **Constraints:** What is forbidden/allowed
- **Steps:** Numbered execution plan
- **Success Criteria:** How to verify

### Phase 5: Context Update

- Update `/.agents/memory-bank/active-context.md`
- Append to `/.backlog/[task].md` if needed

## Verification

- [ ] Human approved handoff
- [ ] Context updated
- [ ] Ready for Build Mode
